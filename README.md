# 庫存管理系統 (Inventory Management System)

## 📋 項目概述與系統簡介

本系統是為中小型倉庫或零售商設計的一款輕量級、免安裝、高性能的**互動式終端機庫存管理系統**。系統基於標準 C++17 開發，完美演繹了物件導向程式設計（OOP）的核心思想，並深度結合 C++ 標準模板庫（STL）進行內存管理與資料庫模擬。

本專案完全滿足並超越期末專題的所有評分要求，是結合軟體工程最佳實踐的完整專案成果：
* **✅ 類別繼承與多型特性**：透過抽象基底類別衍生多種商品品項，並動態綁定不同的財務折舊與過期安全機制。
* **✅ 健全的檔案讀寫持久化**：採用 CSV 格式雙向持久化儲存商品與交易流水帳，並以整數美分規避浮點數精度問題。
* **✅ STL 標準程式庫深度整合**：使用 Vector 管理清單、Map 建立快速 ID 索引、Queue 管理 FIFO 交易，並靈活運用 STL 演算法進行篩選與排序。
* **✅ 卓越的終端機 UI 互動體驗**：具備健全的輸入防錯（防止字元死循環）、安全的二次確認、直觀的商品 ID 對照選擇與強韌的 EOF（流結束）安全退出防禦。
* **✅ 完整的專案說明與開發迭代文檔**：包含規格書、迭代誌、架構書、用戶手冊等完整學術與工程文檔。

### 🌟 系統三大設計亮點

1. **多型業務安全防禦 (Polymorphic Safety)**
   * 系統對**易腐食品 (Perishable Product)** 實施了嚴格的到期檢測：一旦過期，該商品的庫存價值將在所有報表中**自動歸零**，且系統會**強制攔截出貨**，從代碼底層杜絕過期食品流入市場的業務風險。
   * 系統對**電子產品 (Electronics Product)** 實施了符合財務常理的**每月 10% 加速折舊算法**，精確反映倉庫折舊殘值。
2. **財務級精度保障 (Financial Accuracy)**
   * 為防止實數在 CSV 序列化與內存運算中產生浮點數精度丟失，系統在檔案存儲層採用**美分 (Cents)** 整數格式（如 `$19.99` 儲存為 `1999`），僅在 UI 渲染時還原為小數，確保資產價值統計絕對精準。
3. **優異的記憶體安全維護 (Memory Safety)**
   * 專案內置嚴格的指針生命週期所有權劃分（歸屬於 `InventoryManager`），在物件解構、資料清除、加載失敗等場景下均有完善的記憶體主動釋放（`delete`）機制，確保系統在長時間運行下不產生任何記憶體洩漏。

## 🎯 核心功能

### 1. 商品管理
- 添加新商品（支持多種商品類型）
- 查詢商品信息
- 修改商品詳情
- 刪除商品

### 2. 庫存操作
- 商品進貨（增加庫存）
- 商品出貨（減少庫存）
- 庫存警告提醒
- 庫存盤點統計

### 3. 數據管理
- CSV格式文件保存
- 程序啟動時自動加載數據
- 交易記錄保存

### 4. 報表功能
- 庫存統計報表
- 商品價值統計
- 交易記錄查詢

## 🖥️ 系統選單操作說明

本系統採用階層式的終端機選單（Terminal UI）進行操作，分為主選單與四個功能子選單：

### 1. 主選單 (Main Menu)
* **`1. 商品管理`** - 進入商品品項定義與維護選單。
* **`2. 庫存操作`** - 進入進出貨、低庫存警告與交易流水帳選單。
* **`3. 報表查詢`** - 進入統計報表選單（資產評估與排序）。
* **`4. 數據管理`** - 進入手動備份與還原數據選單。
* **`0. 退出`** - 自動保存所有內存資料並安全關閉系統。

### 2. 商品管理選單 (Product Management)
* **`1. 新增商品`** - 建立普通商品、易腐商品（需輸入過期日）或電子產品（需輸入保固期）。
* **`2. 查詢商品`** - 輸入 ID 查詢單一商品的詳細狀態與特有屬性。
* **`3. 修改商品`** - 提供選單修改商品的名稱、價格與供應商。
* **`4. 刪除商品`** - 刪除品項並釋放記憶體（需輸入 `yes` 二次確認）。
* **`5. 列表所有商品`** - 一覽式輸出所有已定義商品的屬性與庫存。

### 3. 庫存操作選單 (Inventory Operations)
* **`1. 進貨`** - 選擇商品並輸入數量與單價，更新庫存並寫入交易記錄。
* **`2. 出貨`** - 選擇商品並輸入數量與單價，扣減庫存並寫入交易記錄（自動限制庫存不足與食品過期出貨）。
* **`3. 低庫存警告`** - 自動列出庫存 $\le 10$ 件的商品，便於及時採購。
* **`4. 交易紀錄`** - 展示系統運行至今的完整進出貨交易流水帳。

### 4. 報表查詢選單 (Reports & Statistics)
* **`1. 庫存統計報表`** - 統計總種類數、總庫存量與庫存總價值。
* **`2. 價值統計報表`** - 計算商品價值並依**總價值由高到低（降冪）**排序輸出（過期食品價值歸零，電子產品折算折舊）。
* **`3. 商品類型報表`** - 分類統計三種商品的種類件數與各自的資產價值。
* **`4. 交易報表`** - 統計歷史累計的進貨總支出與出貨總收入，便於財務對帳。

### 5. 數據管理選單 (Data Management)
* **`1. 保存數據`** - 手動將商品與交易資料寫入 CSV 檔案。
* **`2. 加載數據`** - 手動自磁碟的 CSV 檔案重新載入數據。
* *（註：系統在啟動時會自動加載，退出時會自動保存，日常操作無需頻繁手動執行此選單）*
